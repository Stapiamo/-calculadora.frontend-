Descripción general de Calculadora:
Calculadora es un dispositivo el cual nos permita crear cálculos aritméticos, La función principal de este dispositivo es realizar cálculos, cuentas u operaciones. Con una calculadora básica se pueden realizar operaciones sencillas, tal como suma, resta, multiplicación y división.
